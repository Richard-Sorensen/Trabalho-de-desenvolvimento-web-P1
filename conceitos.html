Para enteder e colocar em pratica os desafios dos exercicios, abaixo tem os conceitos basicos de Python.

  • print( )

    A funcao print() e usada para mostrar mensagens ou valor de variaveis na tela. E ela e funtametal para ver o resultado do codigo

  • input( )

    A funcao input( ) , podemos pedir que o usuario digite alguma informacao. E essa informacao sera considerada um str.

  • Tipos de dados e Conversao

    • str

      Usado para representar textos. Qualquer coisa entre aspas simples ou duplas e considerada uma string

    • int

      Usado para transformar uma str em um numero inteiro.

    • float

      Usado para transformar uma str em um numero decimal.

    • bool

      Usado para represenar valores como verdadeiro ou falso.

  • Opoderacoes Aritmeticas

    • +

      Adicao

    • -

      Subtracao

    • *

      Multiplicacao

    • /

      Divisao

    • //

      Divisao inteira

    • %

      Resto da divisao

  • Ordem de Precedencia

    O python segue a mesma ordem matematica para resolver as operações matematicas. A ordem e:

    1. Parenteces ( )

    2. Exponenciacao **

    3. Multiplicacao * , Divisao / , Divisao inteira // , Resto %

    4. Soma + , Subtracao -
